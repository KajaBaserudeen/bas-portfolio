# Bas Portfolio

Modern portfolio website built with React.js, Tailwind CSS, and Framer Motion.

## Features

- Responsive Design
- Dark Theme
- Smooth Animations
- Project Showcase
- Skills & Experience
- Resume Download
- SEO Optimized

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- Framer Motion
- React Icons

## Installation

npm install
npm run dev

## Build

npm run build
